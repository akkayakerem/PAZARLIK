# Firestore Database'i Bulma

## Yöntem 1: Sol Menüden

1. Firebase Console'da sol menüye bakın
2. Şu seçenekleri arayın:
   - **Firestore Database**
   - **Cloud Firestore**
   - **Firestore**
   - **Database** (genel)

## Yöntem 2: Build Menüsü Altında

1. Sol menüde **Build** yazısına tıklayın
2. Açılan menüde **Firestore Database** seçeneğini arayın

## Yöntem 3: Doğrudan URL

1. Tarayıcınızda şu URL'yi açın:
   ```
   https://console.firebase.google.com/project/trading-app-1447a/firestore
   ```
2. Eğer "Create database" butonu görüyorsanız, Firestore henüz oluşturulmamış demektir
3. "Create database" butonuna tıklayın

## Yöntem 4: Project Overview'dan

1. Firebase Console'da üstte **Project Overview** (veya proje adı) yazısına tıklayın
2. Açılan menüde **Firestore Database** seçeneğini arayın

## Eğer Hala Göremiyorsanız

Firestore henüz oluşturulmamış olabilir. Bu durumda:

1. Firebase Console'da herhangi bir sayfada üstteki arama kutusuna **"firestore"** yazın
2. Veya doğrudan şu URL'yi kullanın:
   ```
   https://console.firebase.google.com/project/trading-app-1447a/firestore/data
   ```

## Alternatif: Realtime Database Kullanma

Eğer Firestore'u bulamıyorsanız ve Realtime Database varsa, kodu Realtime Database kullanacak şekilde değiştirebiliriz. Ama bu büyük bir değişiklik olur.
