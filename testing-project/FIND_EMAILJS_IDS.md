# EmailJS ID'lerini Bulma

## Template ID'yi Bulma

Template ID'yi bulmanın birkaç yolu var:

### Yöntem 1: Template Listesinden
1. EmailJS Dashboard → **Email Templates**
2. Template'inizin yanında **3 nokta (⋮)** menüsüne tıklayın
3. **View** veya **Edit** seçin
4. URL'ye bakın: `https://dashboard.emailjs.com/admin/template/XXXXXXXXX`
   - Buradaki `XXXXXXXXX` kısmı Template ID'nizdir

### Yöntem 2: Template Edit Sayfasından
1. Template'inizi açın (Edit Content butonuna tıklayın)
2. Tarayıcının adres çubuğuna bakın
3. URL'de `template/` sonrasındaki kısım Template ID'dir

### Yöntem 3: Template Ayarlarından
1. Template'inizi açın
2. Sağ üstte **Settings** (⚙️) ikonuna tıklayın
3. Template ID orada görünebilir

## Service ID'yi Bulma

1. EmailJS Dashboard → **Email Services**
2. Gmail servisinizin yanında **3 nokta (⋮)** menüsüne tıklayın
3. **View** seçin
4. URL'ye bakın: `https://dashboard.emailjs.com/admin/integration/service/XXXXXXXXX`
   - Buradaki `XXXXXXXXX` kısmı Service ID'nizdir

## Public Key'i Bulma

1. EmailJS Dashboard → Sol menüden **Account** → **General**
2. **Public Key** bölümünde key'i göreceksiniz
3. Kopyalayın

## Hızlı Kontrol

Eğer hala bulamıyorsanız:
- Template ID genellikle `template_` ile başlar (örn: `template_abc123`)
- Service ID genellikle `service_` ile başlar (örn: `service_xyz789`)
- Public Key genellikle harf ve rakamlardan oluşur (örn: `abc123xyz789`)
