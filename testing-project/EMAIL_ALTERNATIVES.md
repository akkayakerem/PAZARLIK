# Email Gönderme Alternatif Çözümler

## Seçenek 1: 2-Step Verification Açma (Önerilen)

App Passwords için 2-Step Verification açık olmalı:

1. https://myaccount.google.com/security
2. **2-Step Verification** → **Get started** veya **Turn on**
3. Telefon numaranızı doğrulayın
4. Sonra **App passwords** görünecek

## Seçenek 2: EmailJS Kullanma (Daha Kolay, Ücretsiz)

EmailJS kullanarak Gmail App Password olmadan email gönderebiliriz:

1. [EmailJS](https://www.emailjs.com/) hesabı oluşturun (ücretsiz)
2. Gmail servisi ekleyin
3. Email template oluşturun
4. API key alın

Bu yöntem daha kolay ve App Password gerektirmez!

## Seçenek 3: Firebase SMTP Ayarları

Firebase Console'da SMTP ayarlarını kullanabiliriz ama bu da Functions gerektirir.

## Öneri

EmailJS kullanmak en kolay çözüm. İsterseniz EmailJS ile devam edelim?
